# Porn Scraper

### Development
Install Poetry

- macOS
  > brew install poetry

- Linux, macOS, Windows (WSL)
  > curl -sSL https://install.python-poetry.org | python3 -

- Windows (Powershell)
  > (Invoke-WebRequest -Uri https://install.python-poetry.org -UseBasicParsing).Content | py -


Install Dependencies
> poetry install
